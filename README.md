## 🚀 Featured Projects

### 🛡️ SafeZone — Real-Time Civilian Safety Platform
> **PHP · MySQL · Leaflet.js · Bootstrap · JavaScript**

A full-stack web application that provides **real-time, province-level threat alerts** and **interactive shelter mapping** for civilians across Sri Lanka during emergencies.

| Feature | Description |
|---|---|
| 🚨 Live Alerts | Region-filtered threat cards with Warning / Danger / Critical severity |
| 🗺️ Interactive Map | Leaflet.js map with clickable safe-zone markers across Sri Lanka |
| 🏥 Shelter Directory | Live capacity tracking with occupancy bars and photo modals |
| 👤 Civilian Accounts | Province-based personalised alert feed |
| 📢 Incident Reporting | Civilians submit reports; admin reviews before publishing |
| ⚙️ Admin Dashboard | Full CRUD for alerts, zones, and civilian reports |

**Tech highlights:** 3-tier architecture · PDO prepared statements · Session-based role auth · JSON-encoded map data
